# aws-cloud-practitioner-exam-cheat-sheet
Este repositório consiste em uma "Cheat Sheet" (folha de cola) para o exame AWS Cloud Practitioner, fornecendo uma visão resumida dos serviços AWS cobrados no exame. É uma referência rápida e fácil de usar para revisar os principais conceitos e serviços essenciais da AWS, organizados por domínios relevantes para facilitar o estudo antes da prova.
